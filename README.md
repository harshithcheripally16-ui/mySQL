# MySQL Course Tasks

This repository contains a complete MySQL learning course in task format. It is built for practice, revision, and concept-by-concept understanding through executed MySQL commands and their outputs.

The focus of this repo is tasks only. Each task file contains practical SQL work, including table creation, data insertion, queries, updates, deletes, filtering, sorting, and result sets. Anyone opening this repository should be able to read the task files and understand what concept is being practiced, what command was used, and what output MySQL returned.

## Repository Purpose

This repo is meant to help learners understand MySQL by doing. Instead of only explaining syntax, the files show real command-line MySQL sessions with:

- SQL statements
- Inserted sample data
- Query results
- Common mistakes and corrected commands
- Practical examples for each concept

Notebook files are kept local and are not part of the GitHub course material. The GitHub repo is intended to contain the task text files and documentation.
